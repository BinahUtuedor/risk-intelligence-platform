# Dashboard Guide – Risk Intelligence Platform

## Overview

The Risk Intelligence Dashboard provides a simple, interactive interface for exploring country-level risk scores, screening results, and machine learning outputs generated by the Risk Intelligence Platform.

The dashboard is built using **Streamlit** and is intended for compliance professionals, risk analysts, and business stakeholders who require quick access to risk insights.

---

## Purpose

The dashboard enables users to:

- Monitor overall risk exposure
- Identify high-risk countries
- Review model outputs
- Explore processed datasets
- Support compliance and risk-management decisions

---

## Technology

The dashboard is implemented using:

- Python
- Streamlit
- Pandas

Source file:

```text
dashboard/dashboard.py
```

---

## How to Run

From the project root directory:

```bash
streamlit run dashboard/dashboard.py
```

Once the application starts, open your browser and navigate to:

```text
http://localhost:8501
```

## Dashboard Components

### 1. Dashboard Title

The dashboard header displays:

```text
Risk Intelligence Dashboard
```

This provides a clear entry point into the platform and communicates the purpose of the application.

Code:

```python
st.title("Risk Intelligence Dashboard")
```

---

### 2. Key Metrics

The dashboard provides high-level summary metrics designed for executive consumption.

#### Total Records

Displays the total number of countries/entities available in the processed dataset.

Code:

```python
st.metric(
    "Records",
    len(df)
)
```

Business value:

- Indicates dataset coverage
- Shows how many countries were analysed
- Provides context for overall risk exposure

---

#### High Risk Cases

Displays the number of countries classified as high risk.

Logic:

```python
risk_flag == 1
```

Code:

```python
st.metric(
    "High Risk Cases",
    len(
        df[
            df["risk_flag"] == 1
        ]
    )
)
```

Business value:

- Highlights investigation priorities
- Helps compliance teams focus resources
- Provides an immediate overview of risk concentration

---

### 3. Risk Distribution Chart

The dashboard includes a visual representation of risk scores.

Code:

```python
st.bar_chart(
    df["risk_score"]
)
```

The chart displays:

- Distribution of risk scores
- Relative risk levels across countries
- Concentration of high-risk entities

Business value:

- Quickly identifies risk outliers
- Supports trend recognition
- Provides an intuitive visual summary

---

### 4. Data Preview Table

The dashboard displays a sample of the processed risk dataset.

Code:

```python
st.dataframe(
    df.head(100)
)
```

Displayed information may include:

| Column | Description |
|----------|------------|
| country | Country name |
| country_code | ISO country code |
| gdp | Gross Domestic Product |
| tax_revenue_pct | Tax revenue as % GDP |
| sanction_risk | Sanctions flag |
| pep_risk | Politically exposed risk |
| high_risk_jurisdiction | Jurisdiction risk flag |
| risk_score | Composite risk score |
| risk_flag | High-risk classification |

Business value:

- Enables detailed inspection of records
- Supports validation of model outputs
- Facilitates compliance review activities

## Current Limitations

The current dashboard is intentionally lightweight and designed as a proof-of-concept.

Limitations include:

- No filtering functionality
- No search capability
- Static visualisations
- No historical trend analysis
- Country-level analysis only
